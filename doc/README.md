# Documentation

## Value types:
* `int` - Number without dot
* `float` - Number with dot (Floating point)
* `string` - String value
* `none` - Single value of nothing
* `array` - Array of data
* `dict` - Dictionary
* `func` - Function
* `atom` - Atom Value (String alternative). Used for calls and other programming stuff
